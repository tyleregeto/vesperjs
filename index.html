Vesperjs

A library for adding mention auto completion/insertion, ie @username #tag :emoji

View the Project on GitHub tyleregeto/vesperjs

Getting started

<div id="myEditableContent" contentEditable></div>

<script type="text/javascript">
te.newMentionListener('myEditableContent', {
    data: ['homer', 'marge', 'maggie', 'lisa', 'bort'],
});
</script>

Notable features

  • No dependencies. Zero.
  • Only 2 KB (compressed)
  • IE9+ compatibility
  • Works with contentEditable and textareas
  • Pluggable - easily add additional support and custom views
  • Trigger character is configurable, not just '@'
